## What is ClickSay?

ClickSay turns UI feedback into structured, AI-ready prompts in seconds.

1. **Click** any element on a web page
2. **Speak** your feedback naturally
3. **Paste** the structured prompt into Claude, ChatGPT, Cursor, or any AI tool

What takes ~1 minute of typing takes ~8 seconds with ClickSay.
